## рҹ“Ң Overview

**Play TN** is a web-based sports management platform designed to simplify sports event organization across Tamil Nadu. It provides a centralized platform where players, tournament organizers, coaches, and venue owners can manage sports activities efficiently.

The platform aims to digitize the sports ecosystem by enabling users to discover tournaments, manage events, and streamline sports-related operations through an easy-to-use web interface.

## рҹҡҖ Installation

### Clone the Repository

```bash
git clone https://github.com/raghavjr10/play-tn.git
```

### Navigate to Project

```bash
cd play-tn
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
node server.js
```

or

```bash
npm start
```

The application will start on:

```
http://localhost:3000
```

*(Port may vary depending on your configuration.)*
